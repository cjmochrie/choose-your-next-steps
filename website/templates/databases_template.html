Intro to Databases

Overview

Karl Krueger's intro to relational databases was a very quick intro, but got me started in using postgresql on a paractical basis. Some of the topics included were:

  • SQL stored data tpes
  • Python API for SQLite and postgresql
  • The idea of a transaction and the requirement to 'commit' changes to a db
  • Escaping database inserts with data to avoid injection attacks using the Python library Bleach
  • The concept of normalized forms and building larger database structures using foreign keys
  • Using the different types of joins and self joins - I wish this section was longer and with more practice. Joins are tricky
  • A brief look at subqueries and views

The final project was required me to implement several functions to interact with a database to support a Swiss Bracket tournament. It would have been neat to have more testing code available to make it a little easier to implement additional functionality, but otherwise was a cool practical application of SQL.
